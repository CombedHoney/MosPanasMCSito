Chi Siamo
MosPanas è un server Minecraft Java Edition, nato il 18/02/2023. Nato per il medesimo gioco minecraft, dopo una serie di decisioni il server si è trasformato in un server per parlare a tema "Anime" chiamato look sky, ma dopo alcuni problemi siamo tornati su minecraft col nome Look Sky, ma per la nostalgia l'owner ha ridato al server il suo antico nome "Mos Panas".
